Enterprise Cloud Interface Engine
=================================

## About
Enterprise Cloud Interface Engine (ecie) is a solution to securely and reliably connect disperate systems that communicate over differing application and transport protocols.  ECIE is a fully managed cloud solution with code as infastructure so that deployments are consistent and reproducible to minimize human interaction and mistakes. ECIE is specific to AWS, using API Gateway, Lambda, DynamoDB, Cognito, and SQS. An architecture diagram will be provided to demonstrate the method by which message traffic is handled throught the life cycle of a message.
